When using a Dagger SDK in your application:

1. Install the SDK for your preferred language
2. Initialize a Dagger client in your code
3. Use Dagger API methods to implement your workflow
4. Run your application with `dagger run`

The following SDKs are available for embedding Dagger into your application:

- [Go →](/extending/custom-applications/go) Use the Dagger Go SDK to embed Dagger into your Go applications
- [TypeScript →](/extending/custom-applications/typescript) Use the Dagger TypeScript SDK to embed Dagger into your TypeScript applications
- [Python →](/extending/custom-applications/python) Use the Dagger Python SDK to embed Dagger into your Python applications
- [PHP →](/extending/custom-applications/php) Use the Dagger PHP SDK to embed Dagger into your PHP applications
