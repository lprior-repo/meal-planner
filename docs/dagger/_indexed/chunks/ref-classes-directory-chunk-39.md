> **withTimestamps**(`timestamp`): `Directory`

Retrieves this directory with all file/dir timestamps set to the given time.

#### Parameters

#### timestamp

`number`

Timestamp to set dir/files in.

Formatted in seconds following Unix epoch (e.g., 1672531199).

#### Returns

`Directory`
