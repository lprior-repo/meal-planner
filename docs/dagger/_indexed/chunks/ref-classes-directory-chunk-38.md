> **withSymlink**(`target`, `linkName`): `Directory`

Return a snapshot with a symlink

#### Parameters

#### target

`string`

Location of the file or directory to link to (e.g., "/existing/file").

#### linkName

`string`

Location where the symbolic link will be created (e.g., "/new-file-link").

#### Returns

`Directory`
