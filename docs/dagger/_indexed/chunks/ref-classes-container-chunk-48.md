> **withLabel**(`name`, `value`): `Container`

Retrieves this container plus the given label.

#### Parameters

#### name

`string`

The name of the label (e.g., "org.opencontainers.artifact.created").

#### value

`string`

The value of the label (e.g., "2023-01-01T00:00:00Z").

#### Returns

`Container`
