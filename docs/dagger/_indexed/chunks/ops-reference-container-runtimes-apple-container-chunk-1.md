# Apple Container Runtime

> **Context**: Dagger can be used with any OCI-compatible container runtime, including [`container` from Apple](https://github.com/apple/container).


Dagger can be used with any OCI-compatible container runtime, including [`container` from Apple](https://github.com/apple/container).

> **Warning**: The support for Apple's `container` is still early and experimental.
