# Beads Planning Slash Command

This folder contains documentation for the `/pick-up-new-beads-task` slash command used in Claude Code for TDD/TCR-driven development with the Beads issue tracker.

## Files

- `pick-up-new-beads-task.md` - The main slash command definition
- `phases.md` - Detailed breakdown of each development phase
- `quick-reference.md` - Cheat sheet for daily use

## Usage

In Claude Code, invoke with:
```
/pick-up-new-beads-task
```

Or use natural language:
- "pick up task"
- "pick up new beads task"
- "what's ready to work on?"

## Philosophy

This command orchestrates superhuman AI-assisted development through:
- **Strict TDD/TCR** - Test && Commit || Revert
- **Semantic code navigation** - Serena MCP tools
- **Persistent memory** - mem0 for cross-session learning
- **Fractal iteration** - Nested loops for task decomposition
