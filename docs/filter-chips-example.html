3. JavaScript API Examples

The FilterChips object exposes a public API for programmatic control:

// Get current filters
const filters = FilterChips.getFilters();

// Set filters programmatically
FilterChips.setFilters({
  category: ['vegetables', 'fruits'],
  verified: ['true']
});

// Clear all
FilterChips.clearAll();

// Check if active
if (FilterChips.isFilterActive('verified', 'true')) {
  console.log('Verified only is active');
}

// Listen to events
FilterChips.on('filterChange', function(event) {
  console.log('Filter changed:', event);
});

FilterChips.on('filterApply', function(event) {
  console.log('Triggering search with:', event.filters);
});

4. Features

  • Click handlers for filter chips (toggle on/off)
  • Remove filter with X button
  • URL query parameter synchronization
  • Toggle active state visually
  • Automatic search triggering (debounced)
  • Category dropdown with keyboard navigation
  • Full keyboard accessibility
  • Screen reader support (WCAG 2.1 AA)
  • LocalStorage persistence
  • Responsive design (mobile, tablet, desktop)
  • Dark mode support
  • Event system for integration
