# Versioning

> **Context**: import DocCard from '@site/src/components/DocCard';

Scripts, Flows and Apps have unique versions when deployed. You can find the said version any time, fork it or go back to this version.

You can check all the deployed versions and deployment message of a script, flow or app by clicking on the `History` on the script page.
