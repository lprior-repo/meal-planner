## JSON schema resources

Resources of type `json_schema` allow you to store reusable [JSON schemas](./meta-13_json_schema_and_parsing-index.md) that define the structure and validation rules for object parameters. 
They contain a single `schema` field that is a JSON schema object.
These schemas can be referenced as templates when configuring object-type parameters in flows.
