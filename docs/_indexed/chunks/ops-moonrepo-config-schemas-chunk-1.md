# sync config-schemas

> **Context**: The `moon sync config-schemas` command will manually generate JSON schemas to `.moon/cache/schemas` for all our different configuration files.

v1.27.0

The `moon sync config-schemas` command will manually generate JSON schemas to `.moon/cache/schemas` for all our different configuration files.

```
$ moon sync config-schemas
```
