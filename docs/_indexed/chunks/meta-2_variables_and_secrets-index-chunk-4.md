## Contextual variables

[Contextual variables](./meta-47_environment_variables-index.md#contextual-variables) are variables whose values are contextual to the script execution. They are automatically set by Windmill. This is how the [languages](./meta-0_scripts_quickstart-index.md) clients get their implicit credentials to interact with the platform.
