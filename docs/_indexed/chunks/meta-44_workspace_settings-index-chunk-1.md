# Workspace settings

> **Context**: Workspace settings are configuration options specific to each workspace and can be accessed by workspace [admin](../16_roles_and_permissions/index.mdx

Workspace settings are configuration options specific to each workspace and can be accessed by workspace [admin](../16_roles_and_permissions/index.mdx#admin) and instance [superadmin](./meta-16_roles_and_permissions-index.md#superadmin). These settings should not be confused with [Instance settings](./meta-18_instance_settings-index.md).

Each workspace can be configured through the following sections:

- [Users](./meta-16_roles_and_permissions-index.md) - Manage workspace members and their permissions, in particular the operator visibility settings
- [Git Sync](./meta-11_git_sync-index.md) - Configure Git synchronization for your workspace
- [Deployment UI](./meta-12_staging_prod-index.md) - Customize the deployment interface
- [Slack](../../integrations/slack.mdx) / [Teams](../../integrations/teams.mdx) - Set up Slack and Teams integration and notifications
- [Webhook](./meta-4_webhooks-index.md#workspace-webhook) - Configure webhook endpoints and triggers
- [Error Handler](./meta-10_error_handling-index.md#workspace-error-handler) - Define error handling behavior
- [Windmill AI](./meta-22_ai_generation-index.md) - Configure AI-assisted features
- [Object Storage (S3)](./meta-38_object_storage_in_windmill-index.md#workspace-object-storage) - Set up S3-compatible storage
- [Default App](../../apps/1_default_app/index.md) - Configure the default application settings
- [Encryption](./meta-30_workspace_secret_encryption-index.md) - Manage encryption keys and settings
- [Workspace dependencies](./meta-55_workspace_dependencies-index.md) - Centralized dependency management for scripts
- [General](#general)
