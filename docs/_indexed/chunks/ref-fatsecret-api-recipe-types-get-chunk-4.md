## Response

Returns a list of all available recipe types.

| Field | Type | Description |
|-------|------|-------------|
| `recipe_types` | array | List of recipe type names |

### Available Recipe Types

- Appetizers
- Beverages
- Breads
- Breakfast
- Desserts
- Main Dishes
- Salads
- Sandwiches
- Side Dishes
- Snacks
- Soups
