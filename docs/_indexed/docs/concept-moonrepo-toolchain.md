# toolchain

> **Context**: Manage toolchain plugins.

Manage toolchain plugins.

## add

The moon toolchain add [plugin] command will add a toolchain to the workspace by injecting a configuration block into `.moon/toolchain.yml`.

## info

The moon toolchain info [plugin] command will display detailed information about a toolchain, like what files are scanned, what configuration settings are available, and what tier APIs are supported.
