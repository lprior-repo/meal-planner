v1.37.0

The `moon mcp` command will start an [MCP](https://modelcontextprotocol.io) server that listens for requests from AI assistants. This allows for agentic workflows in your favorite editor.

```
$ moon mcp
```

> This command should not be ran manually and instead should be integrated into your editor. View the [MCP guide](/docs/guides/mcp) for more information.
