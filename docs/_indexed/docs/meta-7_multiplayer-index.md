# Multiplayer

> **Context**: <!-- <doc_metadata> <type>reference</type> <category>core_concepts</category> <title>Multiplayer</title> <description>Real-time collaboration on scrip

The Multiplayer feature allows you to collaborate with team members on scripts simultaneously.

    className="border-2 rounded-xl object-cover w-full h-full dark:border-gray-800"

<br/>

> Example of two team members simultaneously working on a script.

<br/>

From a script, just toggle `Multiplayer`, invite other members and start writing scripts in collaboration!

This feature is only available in the [self-hosted Enterprise Edition](/pricing) and the [Windmill cloud](./meta-00_how_to_use_windmill-index.md#windmill-cloud) version.
