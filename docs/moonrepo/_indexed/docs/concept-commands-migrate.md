# migrate

> **Context**: Operations for migrating existing projects to moon.

Operations for migrating existing projects to moon.

## from-package-json

Use the moon migrate from-package-json sub-command to migrate a project's package.json scripts to moon tasks.

## from-turborepo

This command has been deprecated, use the `moon ext migrate-turborepo` command instead.
